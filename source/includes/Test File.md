# DPP DiscoveryBLAH
Discovery/Eureka is the service that binds all DPP services together. 

## Start
Start using systemd service file.

```shell
systemctl start dpp_discovery
```

## Stop
Stop using systemd service file.

```shell
systemctl start dpp_discovery
```
